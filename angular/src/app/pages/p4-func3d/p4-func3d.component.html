Funkcje dwóch zmiennych

Matematycy zajmują się funkcjami wielu zmiennych. Czym jednak one są? Funkcja $g$ jednej zmiennej przyporządkowuje liczbom rzeczywistym liczby rzeczywiste, zapisujemy to w następujący sposób: {{ '$$g:\\mathbb{R}\\to\\mathbb{R}$$' }}

Funkcja $f$ dwóch zmiennych przyporządkowuje parom liczb rzeczywistych liczby rzeczywiste. Oznacza to, iż wartość tej funkcji zależy od dwóch zmiennych. Zapisujemy to w następujący sposób: {{ '$$f:{\\color{blue}{\\mathbb{R}\\times\\mathbb{R} } }\\to{\\color{red}{\\mathbb{R} } }$$' }} W tym wypadku zbiór wszystkich par liczb rzeczywistych {{ '${\\color{blue}{\\mathbb{R}\\times\\mathbb{R} } }$' }} jest dziedziną funkcji $f$, natomiast {{ '${\\color{red}{\\mathbb{R} } }$' }} jest przeciwdziedziną.

Wzór funkcji dwóch zmiennych może być podany bezpośrednio, na przykład: {{ '$$f(x,y)=x^2+y^2$$' }} Wtedy punktowi $(1,2)$ jest przyporządkowana liczba $f(1,2)=5$.

"Modelowanie" funkcji dwóch zmiennych

Wykres funkcji jednej zmiennej jest krzywą. Czym więc jest wykres funkcji dwóch zmiennych? Powierzchnią.

Jest tak dlatego, że dziedziną funkcji jest płaszczyzna. Gdyby płaszczyzną była powierzchnia jakiegoś biurka i w każdym miejscu blatu biurka narysowalibyśmy punkt na wysokości wartości funkcji w tym miejscu, byłaby to jakaś "unosząca się" powierzchnia.

Można wyobrazić sobie to inaczej. W szkole średniej często spotykamy się z tak zwanym parametrem funkcji. Zmieniając parametr, zmienia nam się wykres tej funkcji. Taką właśnie funkcję z parametrem możemy nazywać funkcją wielu zmiennych traktując parametr jako drugą zmienną.

Funkcje zespolone

Funkcja zespolona $f$ jest to przyporządkowanie — liczbom zespolonym przyporządkowujemy liczby zespolone: $$f:\mathbb C\to\mathbb C$$ Jeśli traktować liczbę zespoloną jako parę liczb rzeczywistych jest to funkcja {{ '$$f:\\mathbb{R}\\times\\mathbb{R}\\to\\mathbb{R}\\times\\mathbb{R}$$' }} Oznacza to, że potrzebujemy aż czterech wymiarów aby taką funkcję narysować.

Na tej stronie będziemy mówili głównie o funkcjach holomorficznych. To znaczy takich, które są różniczkowalne. Posiadają one wiele ciekawych własności.

Sposoby przedstawiania funkcji zespolonej

Pierwszym sposobem są tak zwane mapy zespolone. Są to dwa rysunki jakiejś figury, najczęściej kolorowej wraz z wyznaczonymi liniami. Pierwsza jest figurą oryginalną, druga natomiast figurą poddaną przekształceniu (funkcji).

Drugim sposobem jest podzielenie funkcji zespolonej $f$ na dwie funkcje dwóch zmiennych. Każda funkcja zespolona "pobiera", a potem "wyrzuca" dwie liczby rzeczywiste. Gdyby pierwszą funkcję $f_r$ zdefiniować jako część rzeczywistą funkcji $f$, a drugą $f_i$ jako część urojoną funkcji $f$, otrzymamy dwie funkcje pobierające po dwie liczby rzeczywiste, a wyrzucające tylko jedną: $$f_r(x,y)=\Re f(x+yi)$$ $$f_i(x,y)=\Im f(x+yi)$$

Trzecim sposobem jest technika kolorowania dziedziny. Mając płaszczyznę zespoloną każdy punkt kolorujemy zależnie od wartości tej funkcji w punkcie. Najczęściej bierze się pod uwagę argument i moduł wartości. Technika ta jednak jest bardzo skomplikowana, szczególnie dla początkujących osób, które dopiero odkrywają analizę zespoloną.
